# 绯典阁 使用手册

## 1. 启动与首页

打开 绯典阁 后会先进入首页。首页显示当前图鉴壁纸，中间按钮进入系列入口。壁纸可在设置中导入，导入时会先裁切为 16:9，并预览首页按钮遮罩效果。

## 2. 系列入口与图鉴

系列入口用于快速进入不同分类，例如全部角色、某个作品分组、某个作者分组或用户自定义主题。进入图鉴后，顶部工具栏提供搜索、排序、标签筛选、系列快捷筛选、新建、导入和设置。

角色卡片显示主头像、名称、来源和标签。点击卡片会更新右侧预览抽屉；点击抽屉中的立绘区域会打开全屏详情页。

## 3. 详情页与编辑

详情页不再打开单独编辑弹窗。点击详情页顶部的“编辑”后，当前详情页会原位切换为编辑状态，可以修改：

- 名称、来源、别名。
- 分类归属和标签。
- 人物介绍和备注。
- 头像、立绘、语音、模型附件和其他附件。

点击“保存”会写入当前资料库并回到查看状态。点击“取消”会放弃本次编辑；如果是新建角色且未保存，会直接关闭详情页。

## 4. 头像与封面立绘

一个角色可以保存多个头像。头像导入会进入方形裁切器，保存后会自动成为主头像。也可以在头像列表里把任意头像设为主头像。

一个角色可以保存多张立绘。图鉴预览和详情首屏使用“封面立绘”，可以在立绘列表里把任意立绘设为封面。图鉴卡片仍优先显示主头像。

## 5. 图片预览

点击头像或立绘可以打开大图预览。大图预览支持：

- 鼠标滚轮缩放。
- 拖拽平移。
- 双击放大或重置。
- 上一张/下一张切换。
- Esc 关闭。

## 6. 语音与附件

语音是本地资料资产，可以导入、播放、命名并填写台词。模型和其他文件作为附件保存，不再作为 3D 工作台打开。

删除角色会删除当前资料库中的角色文件夹。删除单个资产只会删除当前资料库里的对应文件和 JSON 引用，不会影响导入前的原始文件。

## 7. 设置与资料库路径

桌面端设置页可以查看当前资料库路径、默认资料库路径、配置文件位置，并可切换或打开资料库文件夹。

Windows portable 版如果从 `release` 文件夹启动，会自动使用上级工作区的 `library` 和 `config`。如果把 exe 移动到其他位置，则会使用 exe 所在位置附近的本地资料库。

## 8. Android 端

Android 端使用 Capacitor/WebView 复用同一套界面，但资料库是独立的，存放在应用私有数据目录。Android APK 默认不内置桌面端全量资料库或第三方素材包。

Android 端支持浏览、搜索、详情页内编辑、头像裁切、壁纸裁切、导入立绘、设置主头像和封面立绘。卸载应用会清除 Android 私有资料库，除非后续版本加入导出备份功能。
