# 绯典阁 Design Brief

## Visual Direction

绯典阁 uses a game encyclopedia style. The first screen should feel immersive and character-forward, while the catalog itself stays dense enough for long-term management.

Keywords:

- Immersive
- Collectible
- Fast to scan

## Layout

- Home: full-screen wallpaper, subtle dark overlay, product name, central entry button, small settings access.
- Collections: horizontal/centered collection choices with icons, counts, and short descriptions.
- Catalog: top toolbar, collection shortcuts, rectangular card grid, right-side preview drawer.
- Detail: full-screen overlay with portrait gallery, character facts, description, notes, voice list, and edit controls.
- Editor: practical form layout, asset management sections, save/cancel actions always visible near the top.

## Color And Texture

- Use a dark, game-like base with warm accent colors.
- Avoid a single-hue purple/blue look.
- Let character images carry the strongest color.
- Use translucent panels over visual backgrounds, not heavy nested cards.

## Interaction

- Home entry should feel deliberate and animated.
- Grid cards should have clear hover and selected states.
- The preview drawer slides from the right.
- The full detail view expands from the preview drawer conceptually and uses a larger visual stage.
- Lightbox uses overlay close, Esc close, and previous/next navigation.

## Density

- Home is spacious.
- Collections are medium density.
- Catalog grid is compact, with stable card dimensions.
- Detail is rich but grouped into predictable bands: visual assets, profile text, voices, attachments, notes.
