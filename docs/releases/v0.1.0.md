# 绯典阁 0.1.0 Release Notes

绯典阁 0.1.0 is the first open-source-ready build of the local-first ACG character catalog.

## Highlights

- Game-catalog style home page and catalog flow.
- Custom wallpaper with crop preview and opacity control.
- Collection shortcuts based on selectable tag rules.
- Searchable and sortable character grid.
- Right-side preview drawer and fullscreen detail view.
- Inline detail editing instead of a hidden modal editor.
- Multiple avatars and portraits, with main avatar and cover portrait selection.
- Zoomable and draggable image preview.
- Local voice, model, and attachment metadata management.
- Windows portable package workflow.

## Verification

Local verification before release:

```bash
npm run prepare:open-source
npm run package:win
npm audit --audit-level=critical
```

Expected checks:

- Open-source audit passes.
- Build passes.
- Windows portable exe is generated under `release/`.
- No critical npm vulnerabilities are reported.

## Data Boundary

The release does not include user libraries, private settings, generated build directories, downloaded third-party asset packs, or API keys.

Users maintain their own local `library/` and `config/` directories.
